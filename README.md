# Monte Carlo Simulation Project

This project performs Monte Carlo estimation using empirical resampling from a CSV file.

## Features
- Select CSV file via file explorer
- Compute probability P(X > 50), expected value E[X], and bootstrap CI for E[X]

## How to Run in Visual Studio
1. Open Visual Studio and create a Python environment.
2. Extract this zip file.
3. Open `main.py` in Visual Studio.
4. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
5. Run `main.py`. A file dialog will appear to select your CSV.
